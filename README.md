The user will need to download the file and place it inside xampp/htdocs folder to ensure that it is working in their local host environment. 

After that, open the Xampp Control panel and click start on Apache and Mysql until t turns green. 

In the browser bar, type : http://localhost/phpmyadmin/ and hit Enter. 

On the left hand side bar, click on your sneaker_store database or you can import the database.sql file through the import handling. (optional: download the seed data file: sneaker_store.sql and import to use premade data for testing)  

In another web, type:  http://localhost/sneakers_store/  

The homepage index.php should load
